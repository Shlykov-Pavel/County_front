# Северное графство — концепция проекта

Сайт питомника собак «Северное графство».

## Бренд

- **Название:** Северное графство
- **Породы:** кавалер-кинг-чарльз-спаниели и пудели (основные); ранее были и другие.
- **Характер бренда:** тёплый, семейный, по-доброму — но с нотой английской усадебной элегантности.
- **Ассоциации с названием:** север, зима, снег, морозная синева. Образ — «тёплый дом посреди зимы»: холодная зимняя база + тёплые домашние акценты (как светящееся окно усадьбы в снежных сумерках).
- **Нарратив:** благородный домашний питомник в духе английского зимнего графства. Кавалер-кинг-чарльз — английская королевская порода (назван в честь короля Карла), что усиливает аристократический мотив.

## Стиль: «Зимнее графство»

### Палитра

| Роль | Цвет | HEX |
|------|------|-----|
| Основной (шапка, подвал, заголовки) | Глубокий полночно-синий | `#23364B` |
| Фон | Снежно-белый (чуть голубоватый) | `#F2F5F8` |
| Акцент (кнопки, герб, рамки) | Тёплое золото / латунь | `#C9A227` |
| Дополнительный | Морозный сине-серый | `#8CA3B8` |
| Дополнительный | Ледяной голубой (фоновые блоки) | `#C7D6E2` |

Принцип: холодная сине-снежная база + тёплый золотой акцент, чтобы было благородно, но не холодно.

### Типографика

- **Заголовки:** элегантный засечный «editorial» шрифт — Playfair Display или Cormorant Garamond.
- **Текст:** чистый гротеск — Inter или Nunito Sans.

### Фотостиль

Тёплые, живые фото собак в естественном свете на снежном/светлом фоне. Контраст тёплого рыже-белого окраса (бленхейм) на холодном фоне.

### Детали и формы

- Скруглённые углы у карточек и кнопок (мягкость, дружелюбие).
- Много воздуха, не перегружаем.
- Геральдика: герб/эмблема со щитом, силуэтом спаниеля, короной (отсылка к King Charles), зимний мотив (снежинка/еловая ветвь). Синий + золото — классическое геральдическое сочетание.
- Тонкие золотые линии-виньетки в духе старых родословных грамот.

## Технологии

- **Репозиторий:** monorepo (`backend/` + `frontend/`).
- **Бэкенд:** Go, без лишних фреймворков — стандартный `net/http` (роутер из Go 1.22+). БД: PostgreSQL (планируется).
- **Фронтенд:** React + TypeScript. Рассматриваем Next.js ради SEO и оптимизации изображений. Tailwind CSS, mobile-first.

## Структура страниц

### Публичная часть
- Главная (приветствие, лучшие собаки, актуальные помёты, последние новости)
- Каталог собак → карточка собаки (фото-галерея, родословная, титулы, здоровье)
- Помёты / щенки → карточка помёта со списком щенков
- Новости → страница новости
- О питомнике
- Контакты + форма заявки

### Админка (закрыта логином)
- Вход
- CRUD: собаки, помёты, щенки, новости
- Просмотр и обработка заявок

## Модель данных (черновик)

- **Dog** — кличка, порода, пол, дата рождения, окрас, титулы, фото (галерея), описание, родители.
- **Litter (помёт)** — дата рождения, мать (Dog), отец (Dog/внешний), описание, статус (ожидается / есть щенки / продан).
- **Puppy (щенок)** — помёт, кличка/номер, пол, окрас, цена, статус (доступен / забронирован / продан), фото.
- **Request (заявка)** — имя, контакт, текст, ссылка на щенка/помёт (опц.), статус, дата.
- **News** — заголовок, текст, обложка, дата, опубликовано (да/нет).
- **User (админ)** — логин, хэш пароля, роль.
